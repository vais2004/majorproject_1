# MyStyleSpot - An eCommerce Website

MyStyleSpot is fully responsive ecommerce website.
It's built using HTML, CSS, and JavaScript.

#Features:-   
• Sort by Price  
• Product Details Page    
• Wishlist Management     
• Cart Management     
• Address Management     
• User Profile Page      
• Loading & Alerts       
• Search
